Baseline Font size

The style sheet uses a dynamic calculation based on screen size to set the base font size. All subsequent styles are set as a percentage of that baseline - which is applied to the relevant container element

The calculations is font-size: calc( 16px + (24 - 16) * ( (100vw - 400px) / ( 1000 - 400) )) !important; and sets the maximum font size at 24px based on a media query at 1000px @media screen and (min-width: 1000px) { .card {font-size: 24px !important;}
